Amateur Radio Remote Control

Full CAT control
from any browser

WebRig gives you remote control of your amateur radio transceiver directly from Chrome or Edge — no software to install. Program memory channels, control the VFO, monitor S-meter readings and stream audio, all from a browser tab. Works over LAN, VPN, or locally on the radio machine.

No installation Windows · Linux · macOS · ChromeOS Chrome / Edge (Web Serial API) Open source — LA8DKA
